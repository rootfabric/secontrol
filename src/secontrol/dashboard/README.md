# SE Grid Dashboard

Web-дашборд для визуализации состояния гридов Space Engineers в реальном времени.

## Запуск

```bash
# Из корня проекта:
cd /home/yurig/secontrol

# Установить зависимости (если ещё не установлены):
pip install fastapi 'uvicorn[standard]'

# Запустить:
python -m secontrol.dashboard.app
```

Откройте в браузере: **http://localhost:8080**

## Переменные окружения

| Переменная | По умолчанию | Описание |
|-----------|-------------|----------|
| `REDIS_URL` | `redis://127.0.0.1:6379/0` | URL Redis сервера |
| `REDIS_USERNAME` | — | Owner ID (из .env) |
| `REDIS_PASSWORD` | — | Пароль Redis |
| `DASHBOARD_PORT` | `8080` | Порт веб-сервера |
| `DASHBOARD_HOST` | `0.0.0.0` | Хост веб-сервера |

Все переменные берутся из `.env` файла в корне проекта.

## Управление

- **Клик по гриду** — выбрать для визуализации
- **Колесико мыши** — масштабирование
- **Перетаскивание** — панорамирование
- **Авто-обновление** — каждую секунду через WebSocket
