# План: Автономная добыча льда

## Состояние
- **Корабль**: skynet-baza0 (90797522825535709)
- **Позиция**: (98573, 81571, -132278) — у фарпоста
- **Устройства**: Ore Detector (81229268444653216), Nanobot Drill (108214527447014159), Container (136794540280434587)
- **База**: skynet-farpost0 (133900526852669590)

## Алгоритм (каждые 15 минут)

### Фаза 0: Проверка состояния
1. Проверить текущую задачу в Redis (`se:mining:state`)
2. Если контейнер полон → перейти к Фазе 4 (возврат)
3. Если идёт добыча → продолжить добычу
4. Если ищем астероид → продолжить поиск

### Фаза 1: Отстыковка и взлёт
1. Отстыковаться от фарпоста
2. Включить все системы (двигатели, гироскопы, детектор)
3. Выбрать направление полёта (от базы, не к ней)

### Фаза 2: Поиск астероида с льдом
1. Лететь в выбранном направлении через A* или goto
2. Каждые 500м делать скан ore_detector (scan_and_wait)
3. Проверять ore_cells на наличие Ice
4. Если найден лед → перейти к Фазе 3
5. Если нет льда через 5000м → сменить направление
6. Максимум 3 смены направления → вернуться на базу

### Фаза 3: Добыча льда
1. Подлететь к месторождению льда (50м)
2. Включить Nanobot Drill
3. Ждать заполнения контейнера
4. Периодически проверять уровень заполнения
5. Когда полон → выключить бур, перейти к Фазе 4

### Фаза 4: Возврат на базу
1. Лететь к фарпосту через A*
2. Пристыковаться через dock.py
3. Отключить системы
4. Записать состояние "done" в Redis

## Состояния в Redis

Ключ: `se:mining:state`
Значения:
- `idle` — готов к работе
- `undocking` — отстыковывается
- `searching` — ищет астероид
- `mining` — добывает ресурс
- `returning` — возвращается на базу
- `docking` — стыкуется

Ключ: `se:mining:target` — координаты текущей цели
Ключ: `se:mining:direction` — текущее направление поиска
Ключ: `se:mining:ice_found` — найден ли лед
